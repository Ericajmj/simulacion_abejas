## APUNTES GO 


### INICIAR PROYECTO GO 

```go mod init nombre```

nombre de proyecto, sin acentos, minusculas y guiones para separar 
 1er archivo para crear es main.go en donde se encuentra funcion main, 1ero que se ejecuta.

iniciar repositorio. 
```git init```

### EJECUTAR EL PROGRAMA

comando ```go run main.go```
